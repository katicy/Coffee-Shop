Бизнес-контекст
Сеть кофеен хочет проанализировать эффективность продаж за первое полугодие 2023 года для оптимизации бизнес-процессов и увеличения выручки.

Цели проекта
1. Анализ эффективности магазинов
   - Выявление лидеров 
   - Сравнение ключевых метрик по локациям

2. Анализ продуктовой линейки
   - Определение топ-товаров и категорий
   - ABC-анализ ассортимента

3. Выявление временных паттернов
   - Сезонность по месяцам
   - Динамика по дням недели
   - Пиковые часы продаж

4. Формулировка рекомендаций
   - Конкретные предложения по оптимизации
   - Расчет потенциального эффекта

Ожидаемые результаты
- Интерактивный дашборд для руководства
- Конкретные бизнес-рекомендации
- Автоматизированная система отчетности
- Основа для принятия стратегических решений
