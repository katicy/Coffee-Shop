О проекте
Интерактивный дашборд для анализа продаж сети кофеен за первое полугодие 2023 года. Проект демонстрирует полный цикл анализа данных от сырых данных до бизнес-инсайтов.

Основные возможности
- Интерактивные фильтры по месяцам и магазинам
- KPI панель с ключевыми метриками
- Многомерная визуализация (6+ типов графиков)
- Выводы и рекомендации

Технологии
- Microsoft Excel (сводные таблицы, формулы, дашборды)
- Интерактивные срезы (Slicers)
- Условное форматирование
- Профессиональная визуализация

Ключевые метрики
- Выручка и динамика продаж
- Средний чек и товарооборот
- ABC-анализ товаров
- Эффективность магазинов
- Временные паттерны продаж

Руководство
1. Скачайте файл Coffee_Sales_Dashboard.xlsx
2. Откройте в Microsoft Excel
3. Используйте фильтры для анализа данных
4. Нажмите "СБРОС" для очистки фильтров

Структура проекта
- `Coffee_Sales_Dashboard.xlsx`

- `Project_Description` - описание целей
