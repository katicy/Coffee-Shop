Источник данных
Датасет содержит данные о продажах сети кофеен за первое полугодие 2023 года.

Структура данных
Всего записей: 149117 

Поля данных:
- `transaction_id` - уникальный ID транзакции
- `transaction_date` - дата транзакции
- `transaction_time` - время транзакции
- `transaction_qty` - количество товара
- `store_id` - ID магазина
- `store_location` - локация магазина
- `product_id` - ID товара
- `unit_price` - цена за единицу
- `product_category` - категория товара
- `product_type` - тип товара
- `product_detail` - детали товара

Добавленные расчетные поля:
- `Выручка` = transaction_qty * unit_price
- `День недели` - день недели транзакции
- `Месяц` - месяц транзакции
- `Время суток` - утро/день/вечер

Объем данных
- Период: январь-июнь 2023 года
- Количество магазинов: 3
- Количество товарных категорий: 3
- Количество товарных позиций: 50+

Особенности данных
- Данные очищены и подготовлены для анализа
